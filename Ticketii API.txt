📘 TICKETII API CONTRACT (v1)
🔐 AUTH
1. Register
Http
POST /api/auth/register.php
Request
JSON
{
  "name": "John Doe",
  "email": "john@mail.com",
  "password": "123456"
}
Response
JSON
{
  "success": true,
  "message": "Registration successful",
  "token": "JWT_TOKEN",
  "user": {
    "id": 1,
    "name": "John Doe",
    "email": "john@mail.com"
  }
}
2. Login
Http
POST /api/auth/login.php
Request
JSON
{
  "email": "john@mail.com",
  "password": "123456"
}
Response
JSON
{
  "success": "Login successful",
  "token": "JWT_TOKEN",
  "user": {
    "id": 1,
    "fullname": "John Doe",
    "email": "john@mail.com",
    "role": "user"
  }
}
3. Send OTP
Http
POST /api/auth/send_otp.php
Request
JSON
{
  "email": "john@mail.com",
  "type": "login"
}
4. Verify OTP
Http
POST /api/auth/verify_otp.php
Request
JSON
{
  "email": "john@mail.com",
  "otp": "123456",
  "type": "login"
}
Response
JSON
{
  "success": "Login successful",
  "token": "JWT_TOKEN"
}
5. Google Auth
Http
POST /api/auth/google_login.php
Request
JSON
{
  "token": "GOOGLE_ID_TOKEN"
}
Response
JSON
{
  "success": true,
  "token": "JWT_TOKEN",
  "user": {
    "id": 1,
    "email": "john@mail.com",
    "name": "John Doe"
  }
}
6. Forgot Password
Http
POST /api/auth/forgot_password.php
Request
JSON
{
  "email": "john@mail.com"
}
7. Reset Password
Http
POST /api/auth/update_password.php
Request
JSON
{
  "token": "RESET_TOKEN",
  "password": "newpassword"
}

⚠️ GLOBAL RULES (IMPORTANT)
1. Authentication
Plain text
All protected routes require:
Authorization: Bearer TOKEN
2. Content-Type
Plain text
Content-Type: application/json
Accept: application/json
3. Error Format
JSON
{
  "success": false,
  "message": "Error message"
}
4. Rate Limits
Plain text
Max 5 requests per minute (auth endpoints)
5. Currency
Plain text
All amounts in NAIRA (NGN)
